# Architecture Overview
The platform follows a modular, API-driven architecture optimized for scalability, performance, and multi-service integration.

## Key Layers
- **Frontend:** Next.js + TypeScript  
- **Backend:** Node.js (Express) + Microservices  
- **Database:** MongoDB + Redis  
- **Authentication:** JWT + OAuth  
- **Hosting:** Vercel (SPA), AWS (Backend), Mongo Atlas (DB)
